# 🚀 Activación de Protocolo - nodeone

## Instrucción para Activar Protocolo

**Copia y pega esto la próxima vez que entres:**

```
activa protocolo nodeone
```

O simplemente di:

```
activa protocolo
```

---

## ¿Qué hace esta instrucción?

Cuando uses esta instrucción, el asistente:

1. ✅ **Lee el protocolo** (`.ai-protocol.md`)
2. ✅ **Lee el resumen funcional** (`RESUMEN_FUNCIONAL.md`)
3. ✅ **Entiende las reglas de trabajo**:
   - Solo hacer lo que indiques
   - `??` = analizar opciones (sin ejecutar)
   - `go` = ejecutar la opción que elijas
   - NO hacer nada sin autorización
4. ✅ **Está listo para trabajar** siguiendo el protocolo

---

## Alternativa: Lectura Completa del Proyecto

Si además quieres que lea TODO el código del proyecto:

```
activa protocolo y lee todo el proyecto nodeone
```

Esto hará que:
- Active el protocolo
- Lea todos los archivos .md
- Lea el código principal (app.py, event_routes.py, etc.)
- Entienda completamente el sistema

---

## Nota

El protocolo está en: `.ai-protocol.md`
El resumen funcional está en: `RESUMEN_FUNCIONAL.md`
