# Código QR de Yappy

## Ubicación del QR
Coloca el código QR de @multiserviciostk en este directorio con el nombre:
`yappy-qr-multiserviciostk.png`

## Ruta completa
`/var/www/nodeone/static/images/yappy-qr-multiserviciostk.png`

## Formato
- Formato: PNG
- Tamaño recomendado: 300x300px o mayor (se ajustará automáticamente)
- El QR debe ser el código estático de @multiserviciostk de Yappy

## Uso
El QR se mostrará automáticamente en la página de pago pendiente cuando el usuario seleccione Yappy.
