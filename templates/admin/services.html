¿Qué es esta pantalla?

Acá administrás el catálogo de servicios que ofrece tu organización: nombre, descripción, precio, planes, si requiere cita, tipo (consultivo / agendable) y asesores.

  • Nuevo servicio: botón superior; podés editar desde cada tarjeta (lápiz).
  • Filtros: estado y búsqueda por nombre o descripción.
  • Citas: si el servicio requiere cita, enlazá tipos de cita y agenda desde Citas o el enlace Asesores en la tarjeta.
  • Multi-tenant: si el catálogo va por empresa, el contexto de organización aplica a este listado.
  • Importar / exportar: CSV o Excel con las mismas columnas; sin id se crean filas; con id de esta organización se actualizan. Categoría por nombre (category_name). Solo administradores pueden importar.
